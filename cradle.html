2. Mapping a ShEX

Example notes on mapping a ShEX schema to a Wikidata data model from the IDEA project.
EntitySchema:E413/Artifact with inscription 


PREFIX wd: <http://www.wikidata.org/entity/>
PREFIX wdt: <http://www.wikidata.org/prop/direct/>
SELECT ?item WHERE {
  {
    ?item wdt:P31 wd:Q1640824 .   # inscription
  }
  UNION
  {
    ?item wdt:P31 wd:Q111326612 . # inscribed object
  }
  UNION
  {
    ?item wdt:P31 wd:Q125323109 . # honorary inscription
  }
  UNION
  {
    ?item wdt:P31 wd:Q125323760 . # sacred inscription
  }
  UNION
  {
    ?item wdt:P31 wd:Q125323565 . # boundary inscription
  }

}

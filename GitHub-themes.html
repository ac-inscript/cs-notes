GitHub Jekyll Themes

My Notes - Guida ai temi Jekyll e come implementarli

Nota: Questi temi Jekyll possono essere utilizzati per personalizzare i tuoi repository GitHub o siti statici. I codici forniti mostrano come configurarli nel file _config.yml.

1Cayman

Descrizione: Un tema minimalista e pulito con un design moderno. Perfetto per progetti di documentazione e portfolio personali.

Configuration (_config.yml):
theme: jekyll-theme-cayman title: My Notes
Link ufficiale:
https://github.com/pages-themes/cayman

📖 Come Usare i Temi

Passo 1: Crea il file _config.yml

Nella root del repository, crea un file _config.yml:

# Seleziona uno dei temi sopra theme: jekyll-theme-cayman title: My Notes description: La mia collezione di appunti

Passo 2: Abilita GitHub Pages

  1. Vai in Impostazioni del repository → Pages
  2. Seleziona il branch (di solito main)
  3. GitHub Pages Build and Deploy se non automatico
  4. Attendi qualche minuto per il deploy

Passo 3: Personalizza il Contenuto

Crea pagine in formato Markdown (.md) o HTML

# Esempio: README.md ## Welcome to My Notes Contenuto della pagina principale...

--- layout: default title: Wikidata to TEI Converter --- # How does it work? It retrieves data from a Wikidata item using the Wikidata API, extracts relevant information such as labels, descriptions, and claims, and then constructs a TEI XML document based on this data. The resulting TEI XML can be displayed on the page and downloaded as a file. # Next steps * Implement a more comprehensive mapping of Wikidata properties to TEI elements. * Multilingual support for items and properties labels.
